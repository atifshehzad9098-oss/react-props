# React Props Job Cards

A simple React project that demonstrates the **React Props concept** by creating reusable **User Job Cards**.
Each card displays information about a job opening such as company logo, job title, job type, salary, and location.

## Project Overview

This project is built using **React** and focuses on understanding how **props work in React components**.
The application dynamically renders multiple job cards by passing different data through props.

## Features

* Reusable React components
* Dynamic job cards using props
* Displays company logo, job title, and job details
* Clean and simple UI
* Beginner-friendly React project

## Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Vite (for project setup)

## Installation & Setup

Clone the repository

```
git clone https://github.com/atifshehzad9098-oss/react-props.git
```

Go to the project directory

```
cd cards_project
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
